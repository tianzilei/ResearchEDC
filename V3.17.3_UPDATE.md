## 如何使用

### 1. 应用 Migration

```bash
# 更新数据库到最新版本
mvn liquibase:update -pl core

# 或使用特定配置文件
mvn liquibase:update -pl core -Dliquibase.propertyFile=src/main/filters/datainfo.properties
```

### 2. 手动执行重复数据清理（如需要）

**PostgreSQL:**
```bash
psql -U clinica -d openclinica -f core/src/main/resources/migration/3.17/sql/Duplicate_Delete_v5.sql
```

**Oracle:**
```bash
sqlplus clinica/password@database @core/src/main/resources/migration/3.17/sql/Duplicate_Delete_v2.sql
```

> ⚠️ **警告:** 重复数据删除脚本会修改数据，请在执行前备份数据库！

## 回滚说明

如需回滚本次更新：

